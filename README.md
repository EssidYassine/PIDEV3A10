# Eventus

A robust event management system built with Java and CSS. Eventus provides a comprehensive platform for creating, organizing, and managing events efficiently.

## 🎯 Features

- Event creation and management
- User-friendly interface
- Event scheduling and organization
- Responsive design

## 🛠️ Technologies

- **Java** (92.7%) - Backend logic and core functionality
- **CSS** (7.3%) - Styling and user interface

## 📋 Requirements

- Java Development Kit (JDK) 8 or higher
- Maven or Gradle for dependency management

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies
3. Run the application
4. Access the event management interface
